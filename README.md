## Setup

1. Clone the repo  
   `git clone https://github.com/Giuseppe552/jobmate-ai.git`

2. Setup the Python backend  
   `cd backend && pip install -r requirements.txt`

3. Start the frontend  
   `cd frontend/jobmate-ui && npm install && npm run dev`
